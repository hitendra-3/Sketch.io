# Whiteboard Frontend (Vite + React + React Router)

Real-time collaborative drawing whiteboard interface built with Vite, React, Tailwind CSS, TypeScript, and HTML5 Canvas.

## Features

- Infinite canvas with zoom and pan.
- Interactive drawing toolbar (custom stroke width, colors, roughness style, shape fills).
- User presence listing and real-time live cursor layers.

## Development

Start the frontend individually (if not using concurrent root script):
```bash
npm install
npm run dev
```
Runs the application on `http://localhost:5173`.
To build the application:
```bash
npm run build
```
Outputs static bundle in `dist/`.
